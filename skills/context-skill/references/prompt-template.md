# Prompt Template

Use this when you want a short repeatable prompt for repo context work.

```text
Use $context-skill for this task.

Task:
[describe the repo context gap or context file to update]

Requirements:
- Inspect existing AGENTS and repo docs first.
- Keep AGENTS concise.
- Prefer updating existing durable docs over creating parallel files.
- Capture commands, paths, boundaries, and conventions concretely.

Output:
- files updated
- main context improvements
- remaining stale or missing context
```
